## Setup

The first thing to do is to clone the repository:

```sh
$ git clone https://github.com/shoyimobloqulov/mathcall.git
$ cd mathcall
```

Once `pip` has finished downloading the dependencies:
```sh
 cd project
 python manage.py runserver
```
